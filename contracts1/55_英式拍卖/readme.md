英式拍卖
英式拍卖(English Auction)是一种常见的拍卖形式，拍卖开始于一个最低价格或起拍价，竞
标者根据自己的意愿竞价，且每次出价必须高于前一个出价，直到没有人愿意再加价为止。
最终，出价最高的竞标者赢得拍卖品。此种拍卖形式竞争激烈，持续时间较长，常见于艺术
品、古董、房产等高价值物品的拍卖。
例子
英式拍卖 vs 荷兰拍卖
拍卖形式 价格趋势 胜出条件 出价次数
英式拍卖 价格逐步上升 出价最高者赢 出价 N 次成交
荷兰拍卖 价格逐步下降 首次出价者赢 出价 1 次成交
