要点：
1. 针对智能合约进⾏多次查询时，可能需要调⽤多个函数和多个合约。
2. 单独调⽤每个函数存在时间戳阻塞问题，导致时间戳可能不同步。
3. 可通过编写名为multicall的合约来聚合多个函数调⽤。
4. multicall合约包含⼀个名为multicall的函数,接受合约地址和调⽤数据作为参数。
5. multicall函数是external view,返回每个调⽤的结果。
6. 在multicall函数内部,通过循环调⽤每个⽬标合约,并将结果存储在字节数组中。
7. 在调⽤每个⽬标合约时,使⽤static call以查询⽅式执⾏函数。
8. 编写完成合约后，可以通过部署测试来验证其功能。
