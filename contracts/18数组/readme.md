1. 数组初始化
• 动态数组：可变⻓度，声明为状态变量。⽰例： uint[] public nums;
• 固定⼤⼩数组：⻓度固定不变，声明后不可扩展或缩⼩。⽰例： uint[3] public
nomsFixed;
• 数组赋初值：在合约部署时初始化数组。例如，动态数组 nums 初始化为 [1, 2, 3] ；固定⼤
⼩数组 nomsFixed 初始化为 [4, 5, 6] 。
2. 数组操作
• 插⼊元素：使⽤ .push() 向动态数组添加元素。固定⼤⼩数组不⽀持 .push() 。
• 访问元素：通过索引直接访问数组元素。⽰例： nums[1] 得到元素 2 。
• 更新元素：直接赋新值到指定索引。⽰例： nums[2] = 777; 。
• 删除元素：使⽤ delete 关键字删除指定索引的元素，该位置值回归默认值（例如 uint 的默认
值是 0 ）。
• 弹出元素：使⽤ .pop() 移除动态数组的最后⼀个元素。
• 获取数组⻓度：使⽤ .length 属性获取数组的当前⻓度。
