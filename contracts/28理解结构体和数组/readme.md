1. 理解结构体和数组
• 学习如何定义结构体 ToDo ，包括任务描述（字符串类型）和完成状态（布尔类型）。
• 学习如何将结构体存储在动态数组 toDos 中，此数组设为公开。
2. 功能实现
• 创建任务：编写 create 函数，输⼊任务描述，将新任务添加到 toDos 数组中，默认完成状态
为 false 。
• 更新任务描述：编写 updateText 函数，输⼊任务索引和新的任务描述，更新指定任务的描
述。介绍两种更新⽅式:直接更新和通过变量更新,讨论其对gas消耗的影响。
• 切换完成状态：编写 toggleCompleted 函数，输⼊任务索引，切换该任务的完成状态。
3. 读取数据
• 虽然 Solidity ⾃动⽣成获取函数，但为了练习，⼿动编写 get 函数，输⼊任务索引，返回任务描
述和完成状态。讨论 memory 和 storage 关键词对gas消耗的影响。
4. 编译和部署
• 指导如何编译智能合约并部署到区块链上。
• 演⽰如何调⽤合约函数，如创建任务、更新任务描述、切换任务完成状态。
