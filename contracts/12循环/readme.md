1. For循环介绍
• 基本语法: 使⽤ for 关键字开始，从 i = 0 开始迭代，直⾄ i < 10 。
• 增量操作: 每次循环结束后，使⽤ i++ 进⾏⾃增。
• 跳过迭代: 使⽤ continue 关键字，如 i == 3 时跳过当前迭代。
• 终⽌循环: 使⽤ break 关键字，如 i == 5 时退出循环。
2. While循环介绍
• 基本语法: 使⽤ while 关键字，循环条件内编写代码，例如 while (j < 10) 。
• ⽆限循环防范: 设置有效的退出条件避免⽆限循环。
3. 循环的效率
• Solidity的特殊考虑: 由于Gas成本,应尽量减少循环次数。
