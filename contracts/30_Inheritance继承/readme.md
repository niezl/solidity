1. 继承基础
• 介绍继承概念：在 Solidity 中，继承允许⼀个合约（⼦合约）使⽤另⼀个合约（⽗合约）的代码和
属性，⽽⽆需重复编写相同的代码。
• 合约⽰例：
◦ 合约A:⽗合约，包含基本函数和属性。
◦ 合约B:⼦合约,继承合约A,并可定制部分函数。
2. 使⽤继承减少代码重复
• 关键字 virtual 的应⽤：在⽗合约的函数声明中使⽤ virtual 关键字，表明这些函数可以被
⼦合约继承和修改。
• 定制函数：
◦ 在⼦合约中，需要修改的函数使⽤ override 关键字标识，显⽰这些函数是对⽗合约中标有
virtual 的函数的重写。
3. 继承的实现
• 声明继承关系：通过在⼦合约声明中使⽤形如 contract B is A 的语法来指定合约B继承⾃合
约A。
• 函数继承与重写：
◦ 继承⾃合约A的函数如 foo 和 bar 在合约B中可以被重写。
◦ 合约B还可以调⽤合约A中未被重写的函数,如⽰例中的 baz 函数。
4. 多级继承
• 扩展⽰例:介绍如何创建⼀个从合约B继承,并进⼀步重写函数 bar 的合约C。
• 深⼊重写:在合约C中, bar 函数被重写以返回不同的值，显⽰多级继承中的函数覆盖。
