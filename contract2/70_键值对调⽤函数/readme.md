关键概念
• 在Solidity中调⽤函数时,通常需要按声明的顺序传⼊参数,这在参数多时会造成记忆负担。
• 可以使⽤键值对调⽤函数，从⽽避免记忆参数顺序的需求。
详细说明
• 标准调⽤⽅法：调⽤函数时必须记住参数顺序。例如，⼀个名为 sumFunc 的函数需要依次传⼊
整数x、y、z,地址a,布尔值b,和字符串c。
• 使⽤键值对调⽤：在调⽤函数时，不需要按照参数声明的顺序传⼊。可以使⽤⼤括号 {} 并标明
每个参数的键值，如 {x: 1, y: 2, z: 3, a: address(0), b: true, c:
'string'} ，这样参数顺序可以任意调整，如 {a: address(0), b: true, c:
'string', x: 1, y: 2, z: 3} 。
优势
• 减少错误：使⽤键值对调⽤函数减少了因顺序错误⽽导致的调⽤失败。
• 提⾼代码可读性：键值对⽅法使代码更清晰，参数的⽬的和传⼊值⼀⽬了然。
